# DNA(Digital Nomad Attraction)
본 서비스는 해외 디지털 노마드를 위한 맞춤형 워케이션 지역 추천 서비스입니다. <br />
사용자의 설문 응답을 기반으로 관광 패턴과 선호도를 분석하여,
개인 취향에 맞는 워케이션 지역을 추천하고 관련 정보를 제공합니다.
